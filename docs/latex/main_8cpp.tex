Este programa implementa un decodificador para el protocolo PRT-\/7, que recibe tramas de tipo LOAD y MAP desde un puerto serial o archivo de simulación. Utiliza una lista doblemente enlazada para almacenar fragmentos decodificados y una lista circular como rotor de mapeo (cifrado César dinámico).\hypertarget{main_8cpp_compile}{}\doxysubsection{\texorpdfstring{Compilación}{Compilación}}\label{main_8cpp_compile}
