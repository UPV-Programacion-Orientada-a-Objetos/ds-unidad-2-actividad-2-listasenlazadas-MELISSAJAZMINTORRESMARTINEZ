Descripción detallada

Decodificador de Protocolo Industrial PRT-7.

Autor
[Tu Nombre]
Fecha
2025
Versión
1.0

Este programa implementa un decodificador para el protocolo PRT-7, que recibe tramas de tipo LOAD y MAP desde un puerto serial o archivo de simulación. Utiliza una lista doblemente enlazada para almacenar fragmentos decodificados y una lista circular como rotor de mapeo (cifrado César dinámico).

Compilación

mkdir build
cd build
cmake ..
make

Uso

./prtdcd --sim entrada.txt
./prtdcd --serial /dev/ttyUSB0
